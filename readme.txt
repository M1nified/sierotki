Sierotki.

== Description ==

Wtyczka zastępuje spacje znajdujące się za sierotkami na jedną twardą spację.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
